# How to run this #

1) Create a folder called CNA

2) git clone https://github.com/leeadh/CNA_Lesson.git

3) cd CNA_lesson_1

4) Run npm init 

# Run the first application to do hello world # 

1) node app_1.js

2) go to localhost:3000

# Run the second application to connect to mongoDB # 

1) node app_2.js

2) go to localhost:3000
